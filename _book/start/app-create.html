创建应用 Create

1. 进入管理中心，点击右上角按钮进入“开发者控制台”

2. 点击右上角“创建应用”创建新应用

  • 应用列表中是您已经创建的应用，您可以点击应用名进入并管理该应用。

3. 在已获得标签中找到阿里大鱼，点击右侧按钮创建应用

4. 输入应用名称，创建应用获得AppKey

5. 创建应用后自动跳转至应用管理页，通过概览可以查看当前应用的APP证书，查看App Key和App Secret

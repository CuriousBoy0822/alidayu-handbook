订阅API返回状态

订阅接口返回状态 Subscribe API Status

1. 进入阿里大鱼>管理中心>开发者控制台

2. 进入需要订阅消息的应用

3. 进入“消息服务—订阅消息”

  • 找到alibaba_aliqin_FcSmsDR订阅短信发送结果报告，找到alibaba_aliqin_FcCallCdr订阅语音呼叫结果报告

# 开发者入驻

## 1. 访问阿里大鱼首页，点击“加入阿里大鱼”
- 使用淘宝和1688账号（昵称）登录、或者使用支付宝账号（邮箱或手机）、支持阿里云账号（邮箱）登录。   

**注意：您用支付宝/阿里云账号首次登录阿里大鱼平台时，需激活账号，设置会员名。激活成功后需要重新打开阿里大鱼网站，重新登陆，然后按提示完成入驻**
